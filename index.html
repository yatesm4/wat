About me



Welcome, and thanks for visiting my online portfolio. My career in Web Development started in late 2015, when I was just making simple sites for fun as I learned the ins and outs. Now, I have been an intern at THiNC Technologies Inc for 6 months as a Junior Web Developer, and my experience has been wonderful.

My strategy has always been to 'sink or swim' by jumping into something new, with no experience, and teaching myself the means to achieve something great. Whether it be PHP or Javascript, or even delving into C++ and Java. Being a fast learning, cross-functional programmer has always been my greatest strength.

Cavasin Moving

Creating Cavasin Moving's web site was my first experience with professional web development. The site was made through WordPress, with custom design implemented to give it a unique feel. The site, and all it's copy, was done by myself alone.

  • Client: Cavasin Moving
  • Date: Oct 2015
  • Service: Front End Web Development

Validu

Working back end for Validu's site was a great experience for developing custom plugins and features for a large scale site.

My work included: transposing records from a database into a visual display, either through a page or through a generated PDF; creating features that allow users to create/modify records in a database through a web interface; create a server-side service to email individuals upon a criteria-meeting record being submitted to the database via mobile application, as well as an 'auto-flagging' option to mark these database entries for later information processing purposes.

  • Client: Validu
  • Date: December 2016 - March 2017
  • Service: Back End Web Development

MMORPG - Side Project

This is an ongoing side project of mine, to develop a basic MMORPG game with the styling of an old Nintendo Game. It is current in a very bare bones state of development, but anyone is more than welcome to check it out at any point as it is all open source.

  • Source: MMORPG - GitHub Repository
  • Date: March 2017
  • Service: GameMaker Node.js
